Some details

Note that in the constructor of Network, we use randn instead of rand. This generates random numbers from a normal distribution ranging from negative infinity to positive infinity. If we use rand, it will only generate values from 0 to 1. However, since the activation function used here is the sigmoid function, the resulting z (which is pretty large) in the first hidden layer will be mapped close to 1, making the network converge very slowly.
